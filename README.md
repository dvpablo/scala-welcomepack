SCALA-WELCOMEPACK
=================
This is a repo showing a basic setup in SCALA, using:

- [SBT](http://www.scala-sbt.org/) for dependency management and test


##Build
```
$ sbt
$ ./build.sh
```

##Test
```
$ sbt
> test
```

##Run
```
$ sbt
> run
$ ./run.sh
$ ./runScript.sh
```
